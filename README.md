# 🎵 Music Library

A multi-page music library web app built with HTML and CSS. Users can browse a collection of 10 songs, view song details and lyrics, and play audio tracks — all with a sleek dark-themed UI.

## 📁 Project Structure

```
music-library/
├── index.html          # Homepage with music grid
├── music1.html         # It's Alright to Cry
├── music2.html         # Fine Girl
├── music3.html         # Into the Unknown
├── music4.html         # Show Yourself
├── music5.html         # Let It Go
├── music6.html         # What Else Can I Do?
├── music7.html         # The Family Madrigal
├── music8.html         # Waiting on a Miracle
├── music9.html         # Remember Me
├── music10.html        # Un Poco Loco
├── style.css           # Global styles
├── images/             # Album/song cover images (Music1.jpg – Music10.jpg)
└── audio/              # Audio files for each song
```

## ✨ Features

- **Music grid** — 10 songs displayed in a responsive 5-column grid on the homepage
- **Song detail pages** — each song has its own page with cover image, lyrics, and an audio player
- **Navigation bar** — with Genre and Artist filter dropdowns
- **Audio playback** — built-in HTML5 `<audio>` player on each song page
- **Scroll-to-top button** — sticky floating button with a smooth scroll animation
- **Dark theme** — deep navy and dark purple color palette throughout

## 🛠️ Built With

- **HTML5** — semantic structure, multi-page layout
- **CSS3** — CSS Grid, Flexbox, hover effects, keyframe animation
- **Font Awesome 5** — icons (scroll-to-top arrow)

## 🚀 Getting Started

No build tools or dependencies needed. Just open `index.html` in your browser.

```bash
# Clone or download the project, then open in browser
open index.html
```

Make sure the `images/` and `audio/` folders are in the same directory as the HTML files for covers and audio to load correctly.
